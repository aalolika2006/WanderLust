# 🌍 WanderLust

A full-stack travel listing web application where users can create, view, and review travel listings.

---

## 🚀 Features
- User authentication system
- Create and manage travel listings
- Add reviews and ratings
- Image support for listings
- Server-side validation using Joi
- MongoDB database integration
- Cascade delete for reviews when listing is deleted

---

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS templates
- Bootstrap
- Joi (validation)

---

## 📁 Project Structure
- Models (Mongoose schemas)
- Routes (Express routes)
- Views (EJS frontend)
- Public (CSS/JS files)
- Utils & middleware
